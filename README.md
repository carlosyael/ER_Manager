# ER_Manager
Diseñada para gestionar automáticamente los archivos de guardado de Elden Ring. Permite alternar de forma rápida y segura entre el juego base (Vanilla) y el mod Seamless Co-op, automatizando la conversión de extensiones de guardado (.sl2 ↔ .co2) para evitar baneos y gestionar tu progreso sin complicaciones.
